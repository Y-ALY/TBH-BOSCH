## Import Verification Results

A comprehensive grep was run across all `.py` and `.js` files (excluding `.git`, `__pycache__`, `.pytest_cache`) for imports or references to every file listed above:

- **No root-level script is imported by any active code.**
- Root `scanner.py` is NOT imported — all active code uses `src.scanner` (verified via `import scanner` / `from scanner import` search).
- `backend/ai_parser/` is NOT imported by anything outside `backend/`.
- The standalone HTML files (`gdpr_loaders.html`, `gdpr_modal.html`) are NOT referenced by any template — templates use `templates/gdpr_modal_snippet.html` instead.
- The JS files (`clean_script.js`, etc.) are NOT referenced by any HTML or other JS.

**Conclusion:** All files proposed for moving are safe to relocate; none are imported by the running application.
